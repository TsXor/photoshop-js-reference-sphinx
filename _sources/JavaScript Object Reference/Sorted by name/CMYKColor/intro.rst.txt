=========
CMYKColor
=========

Defines a CMYK color, used in the SolidColor object.

See also GrayColor, HSBColor, LabColor, NoColor, RGBColor

----------
Properties
----------

.. property:: black
	
	The black color value (as percent).
	
	:Permission: Read-write. 
	
	:Type: :class:`number [0.0..100.00]`


.. property:: cyan
	
	The cyan color value (as percent).
	
	:Permission: Read-write. 
	
	:Type: :class:`number [0.0..100.00]`


.. property:: magenta
	
	The magenta color value (as percent).
	
	:Permission: Read-write. 
	
	:Type: :class:`number [0.0..100.00]`


.. property:: typename
	
	The class name of the referenced CMYKColor object.
	
	:Permission: Read-only. 
	
	:Type: :class:`string`


.. property:: yellow
	
	The yellow color value (as percent).
	
	:Permission: Read-write. 
	
	:Type: :class:`number [0.0..100.00]`
