DocumentPrintSettings¶

The print settings for a document.

Read-write.

Type:

SolidColor

Read-write.

Type:

UnitValue

Read-write.

Type:

boolean

Permission:

Read-write.

Type:

boolean

property colorBars¶

Read-write.

Type:

boolean

Read-write.

Type:

number

Read-write.

boolean

Color handling.

Read-only.

Type:

PrintColorHandling

Read-write.

Type:

string

property flip¶

Read-write.

Type:

boolean

property hardProof¶

Print a hard proof.

Permission:

Read-write.

Type:

boolean

property interpolate¶

Read-write.

Permission:

Type:

boolean

Read-write.

boolean

boolean

Read-write.

boolean

Read-write.

Type:

Intent

property posX¶

Read-only.

Type:

UnitValue

property posY¶

The y position of the image on page.

Permission:

Read-only.

Type:

UnitValue

property printBorder¶

The width of the print border.

Permission:

Read-write.

Type:

UnitValue

property printerName¶

Name of the printer.

Permission:

Read-write.

Type:

string

color space for printer. Can be nothing (meaning same as source); ‘Working RGB’, ‘Working CMYK’, ‘Working Gray’, ‘Lab Color’ (meaning one of the working spaces or Lab color); or a string specifying a specific colorspace (default is same as source)

Permission:

Read-write.

Type:

string

Read-write.

Type:

boolean

number

boolean
