CameraRAWOpenOptions¶

Options for opening a document in Camera RAW format using the Application.open() method.

Properties¶

property bitsPerChannel¶

The number of bits per channel.

Permission:

Read-write.

Type:

BitsPerChannelType

property blueHue¶

The blue hue of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property blueSaturation¶

The blue saturation of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property brightness¶

The brightness of the shot.

Permission:

Read-write.

Type:

number [0..150]

property chromaticAberrationBY¶

The chromatic aberration B/Y of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property chromaticAberrationRC¶

The chromatic aberration R/C of the shot

Permission:

Read-write.

Type:

number [-100..100]

property colorNoiseReduction¶

The color noise reduction of the shot.

Permission:

Read-write.

Type:

number [0..100]

property colorSpace¶

The colorspace for the image.

Permission:

Read-write.

Type:

ColorSpaceType

property contrast¶

The contrast of the shot.

Permission:

Read-write.

Type:

number [-50..100]

property exposure¶

The exposure of the shot.

Permission:

Read-write.

Type:

number [-4.0..4.0]

property greenHue¶

The green hue of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property greenSaturation¶

The green saturation of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property luminanceSmoothing¶

The luminance smoothing of the shot.

Permission:

Read-write.

Type:

number [0..100]

property redHue¶

The red hue of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property redSaturation¶

The red saturation of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property resolution¶

The resolution of the document in pixels per inch.

Permission:

Read-write.

Type:

number [1..999]

property saturation¶

The saturation of the shot.

Permission:

Read-write.

Type:

number [-100..100]

property settings¶

The global settings for all Camera RAW options. Default: CameraRAWSettingsType.CAMERA.

Permission:

Read-write.

Type:

CameraRAWSettingsType

property shadows¶

The shadows of the shot.

Permission:

Read-write.

number [0..100]

Read-write.

number [0..100]

Read-write.

CameraRAWSize

Type:

number [2000..50000]

The tint of the shot.

Permission:

Read-write.

Type:

number [-150..150]

The class name of the referenced cameraRAWOpenOptions object.

Read-only.

Type:

string

Permission:

Read-write.

Type:

number [-100..100]

Permission:

Read-write.

Type:

number [-100..100]

The white balance options for the image. These are lighting conditions that affect color balance.

Permission:

Read-write.

Type:

WhiteBalanceType
